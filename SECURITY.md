# Security Policy

## Supported Versions

RepoJudge is currently in active development. We support the latest major version.

| Version | Supported          | Notes |
| ------- | ------------------ | ----- |
| 1.0.x   | :white_check_mark: | Current Release |
| < 1.0   | :x:                | Alpha/Beta versions |

## Reporting a Vulnerability

We take the security of RepoJudge very seriously. If you have found a security vulnerability, please DO NOT open a public issue.

### How to Report

Please email us directly at **security@repojudge.com**.

In your report, please include:
1.  **Type of issue:** (e.g., SQL Injection, XSS, etc.)
2.  **Steps to reproduce:** Detailed steps to recreate the issue.
3.  **Impact:** How this vulnerability affects the system or users.

### Response Timeline

-   We will acknowledge your report within **48 hours**.
-   We aim to provide a fix or a mitigation plan within **1 week**.

Thank you for helping keep RepoJudge safe! ❤️
