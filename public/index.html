Frequently Asked Questions

Got questions? We've got answers

Is my code safe? Do you store it?

Absolutely. We don't store your code. We fetch it directly from GitHub, send it to Google's Gemini AI for analysis, and return the results. Nothing is saved on our servers. Your code stays on GitHub.

Can I analyze private repositories?

Yes! Login with your GitHub account and grant us read access. We use OAuth with minimal permissions (read-only). You can revoke access anytime from your GitHub settings.

What languages are supported?

RepoJudge supports all major programming languages: JavaScript, TypeScript, Python, Java, Go, Rust, C++, Ruby, PHP, and more. The AI understands context regardless of the language.

Is it really free?

Yes, for public repositories it's 100% free with no limits. For private repositories, you get a generous free tier. We may introduce premium features in the future, but the core analysis will always be free.

How accurate is the AI analysis?

We use Google's Gemini 1.5 Pro, one of the most advanced AI models. It understands code context, not just syntax. While no tool is perfect, our users report catching real bugs they would have missed in manual reviews.
